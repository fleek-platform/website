Deploying AI agents on Fleek has never been a hassle and you can now deploy from within your applications and projects as well. By exposing these APIs, Fleek enables users to seamlessly integrate AI agent management capabilities into their applications.

## Getting a Bearer Token

Fleek supports bearer tokens and API tokens, enabling secure access. API tokens are user-specific and share the same permissions as your account. Follow these steps to obtain your bearer token:
Log in to the Fleek app.

1. Open your browser's developer tools and navigate to the Cookies section.
2. Locate the `accessToken` under `app.fleek.xyz` and copy its value.
3. This token will be required for authentication when creating an API token.
